# 📈 AI Saham Indonesia PRO

AI trading dashboard saham Indonesia berbasis Python + Streamlit.

## Features

- Live Candlestick Chart
- AI Trendline
- Support Resistance
- BUY/SELL Signal
- Compare Saham
- Volume Analysis
- Dark Mode TradingView Style

## Install

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```# ai_saham
